# Sample Inputs – Reinforcement Learning + Role-Based Prompts (main.py)

This program has **two activities**.  
First, you choose the activity by typing **1** or **2**.

---

## Activity Choice

When you see:
```
Choose an activity:
1) Reinforcement Learning
2) Role-Based Prompts
>
```

Type:
- `1` for **Reinforcement Learning**
- `2` for **Role-Based Prompts**

---

## Sample Run 1: Reinforcement Learning (Choice = 1)

### 1) Choose activity
```
1
```

### 2) Enter a prompt
```
Describe the lion.
```

### 3) Rate the response (1 to 5)
```
4
```

### 4) Provide feedback for improvement
```
Add 3 key facts, use simple language, and include one fun fact at the end.
```

## Tip
- In **Reinforcement Learning**, your feedback should be specific (tone, length, format).
- In **Role-Based Prompts**, try the same topic in different categories to see how the “expert role” changes the explanation.
