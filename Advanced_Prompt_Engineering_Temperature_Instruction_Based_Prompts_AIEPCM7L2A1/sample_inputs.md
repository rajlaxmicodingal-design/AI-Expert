# Sample Inputs – Temperature + Instruction Prompts Activity (main.py)

Use these sample inputs when the program asks for:
1) **Creative prompt**  
2) **Topic**  
3) **Your instruction-based prompt**  
4) **Temperature value**  
5) **Bonus streaming choice (y/n)** and a **bonus prompt** (only if you type `y`)

---

## Sample Run 1 (Easy + Fun)

### 1) Enter a creative prompt:
```
Write a short story about a robot who learns to paint.
```

### 2) Choose a topic:
```
space exploration
```

### 3) Enter your instruction-based prompt:
```
Write a 6-step guide for planning a weekend trip. Include a budget, a checklist, and 2 safety tips.
```

### 4) Set temperature (0.1 to 1.0):
```
0.7
```

### 5) BONUS: streaming-like output? (y/n)
```
n
```

---

## Sample Run 2 (School-friendly + Structured)

### 1) Enter a creative prompt:
```
Create a poem about friendship that rhymes.
```

```
Explain climate change for Grade 6 students using: 3 bullet points, 1 real-life example, and 1 short analogy.
```

```
y
```

```
Write a funny 4-line rap about recycling.
```

```
artificial intelligence
```

## Tip
- **Lower temperature (0.1–0.3)** → more predictable and structured output  
- **Higher temperature (0.7–0.9)** → more creative and varied output
