# Sample Inputs – Bias Mitigation + Token Limits Activity (main.py)

This program has **two activities**.  
First, choose the activity by typing **1** or **2**.

---

## Activity Choice

When you see:
```
Choose an activity:
1) Bias Mitigation
2) Token Limits
>
```

Type:
- `1` for **Bias Mitigation**
- `2` for **Token Limits**

---

## Sample Run 1: Bias Mitigation (Choice = 1)

### 1) Choose activity
```
1
```

### 2) Enter a prompt to explore bias
```
Describe the ideal doctor.
```

### 3) Modify the prompt to make it neutral
```
Describe the qualities of a good doctor, focusing on skills and behavior.
```

## Sample Run 3: Token Limits (Choice = 2)

### 1) Choose activity
```
2
```

### 2) Enter a long prompt (paste a long paragraph)
```
Write a detailed story about a young explorer who finds an ancient map in a library, travels across deserts and forests, meets three helpful strangers, solves a riddle to open a hidden temple, and discovers a forgotten city. Include descriptions of the setting, the explorer’s feelings, the challenges faced, and a lesson learned at the end. Also describe the weather, the food they eat, and the tools they use during the journey. Make it engaging and vivid, with a clear beginning, middle, and end.
```

### 3) Condense the prompt to be more concise
```
Write a short adventure story about a young explorer who finds an ancient map, solves a riddle, and discovers a hidden city. Keep it under 200 words.
```

---

## Sample Run 4: Token Limits (Choice = 2) – Different Example

### 1) Choose activity
```
2
```

### 2) Enter a long prompt
```
Explain the water cycle in great detail, including evaporation, condensation, precipitation, runoff, infiltration, groundwater, and transpiration. Include examples from real life, how weather affects each stage, and how humans impact the cycle. Provide at least two examples for each stage and explain why the cycle is important for life on Earth.
```

### 3) Condense the prompt
```
Explain the water cycle for Grade 6 students in 6 bullet points with one real-life example.
```

---

## Tip
- In **Bias Mitigation**, the second prompt should remove stereotypes and focus on neutral qualities.
- In **Token Limits**, the condensed prompt should keep only the most important requirements.
